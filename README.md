##  How to Run

### Backend:
1. Open `Backend-Java` in any Java IDE.
2. Run the `Main.java` file.
3. Use the console to interact with the system.

### Frontend:
1. Open `Frontend/index.html` in our browser.
2. Try out the simulated web version of the system.
